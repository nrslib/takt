# Review Target

## Project

Small TypeScript utility library (`sample-project`). Source lives under
`src/`, tests are Vitest (`src/*.test.ts`).

## Target

The diff under review is provided inline in the review request (User
Request section). Review only that diff; the rest of the repository is
pre-existing context.

## Constraints

- Review-only run: no builds or test execution are required.
- The working directory contains the post-change state of the files.
